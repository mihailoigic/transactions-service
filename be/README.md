## Project setup

```bash
$ npm install
```

## DB Setup

1. Use .env.example and create your own .env for DB credentials setup
2. Run `npm run migration:run` for migrations to run and create `transactions` table

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
